# Net485

Net485 is an protocol to work with the RS485 network universally. 
Compatible with:
- Windows (VisualStudio) / (Qt).
- Linux (Qt).
- MacOSX  (Qt).
- Microchip PIC (MPLAB / MPLABX)
- Arduino (Arduino IDE).
